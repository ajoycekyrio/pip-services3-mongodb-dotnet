Detailed Description

Helper class that resolves MongoDB connection and credential parameters, validates them and generates a connection URI.

It is able to process multiple connections to MongoDB cluster nodes.

Configuration parameters

connection(s): discovery_key: (optional) a key to retrieve the connection from IDiscovery host: host name or IP address port: port number (default: 27017) database: database name uri: resource URI or connection string with all parameters in it credential(s): store_key: (optional) a key to retrieve the credentials from ICredentialStore username: user name password: user password
